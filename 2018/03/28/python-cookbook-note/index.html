Python Cookbook 学习笔记

发表于 2018-03-28 | 分类于 programing

Python是一门下限极低，上限又很高的语言。经验丰富的程序员可以利用python中的各种“奇技淫巧”写出美观又快速的代码，此谓之“Pythonic”。因此特地拜读《Python Cookbook》记心得笔记于此。

itemgetter和attrgetter

itemgetter函数创建一个函数对象，用于获得对象中的某一个元素，常用于max等函数的key参数。

只有自定义类型实现了__getitem__时才可以使用此函数。
对于自定义对象来说，可以使用attrgetter()更方便地获得成员变量。

命名元组

在C中，可以定义一个结构体保存一些变量，结构体中的变量的名可以起到提示作用，而通过namedtuple，python也可以为元组中的每一个元素自定义一个名字。这种方法比使用字典更加高效。

之后可以通过sub.addr和sub.joined访问其中的信息。除此之外，它与普通元组无任何差别，同样支持len、unzip、hashing等操作。

若要修改命名元组中的数据，须使用_replace方法，此方法会创建一个全新的命名元组，原来的保持不变。

合并字典

使用ChainMap可以将多个字典合并成一个。

c看起来就像是包含了dict1，dict2等所有dict中的key的字典，若有重复的key，以先出现的为准，后出现的key所对应的value会被丢弃。

与字典的update方法相比，ChainMap并不会创建一个新的实例对象，而是保存源对象的索引，因此当源字典内容发生改变的时候，ChainMap对象也会随之改变。

ChainMap可以有“作用域”，通过此特性可以临时覆盖某些值，过后在将其恢复。

此特性有些像C中的块作用域，代码块内部可以覆盖外部的同名变量，但是在离开块之后，原变量不受任何影响。

将Unicode统一成规范模式

在unicode中，同一个字符可以被表示成不同的合法代码序列。比如说某些带音调的字母可以有一个单独的编码，也可以表示成一个普通字母加上音调符号的方式。为了处理方便，需要将其统一。

从字符串中去除不必要的字符

对齐字符串

str有ljust、rjust和center三个内置方法用于对齐字符串，分别为左对齐、右对齐和居中对齐。这三个方法都接受一个参数指定对齐后的字符串的长度，和一个可选参数表示用哪个字符来补足。

也可以用format方法进行格式化字符串。

字符串插值

python不直接支持像PHP一样在字符串中直接得到变量的值，但是可以通过format方法来模拟这项awesome的功能。

vars也可以接受一个类实例作为参数，这时它会在类中查找name和n这两个变量。

这个函数的一个缺点就是无法优雅地处理缺少某个值的情况，解决这个问题的办法可以是定义一个带有、__missing__ 的字典类对vars()的结果进行包装。

若要经常执行这样的步骤，则可以将相关的代码隐藏在一个函数中，此技巧称为“frame hack”
