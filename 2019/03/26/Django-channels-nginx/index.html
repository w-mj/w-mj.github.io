Django-Websocket 部署记录

发表于 2019-03-26 | 分类于 default

WebSocket是一种在单个TCP连接上进行全双工通信的协议，它允许服务端主动向客户端推送数据，在网络游戏、聊天室等场合有很大用处[1]。通常情况使用Wsgi部署的Django并不能提供websocket支持，此时需要引入asgi作为服务接口。

如图所示，ASGI将消息按照协议分成http和websocket两类，http消息依然按照Django传统的url-view方式处理，websocket消息则由新的“消费者“机制处理。
