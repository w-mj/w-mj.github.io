UTF-8

  • 对于UTF-8编码中的任意字符B，如果B的第一位为0，则B为ASCII码，并且B独立地表示一个字符。
  • 如果B的第一位为1，第二位为0，则该字符由多个字节组成，且B不是第一个字节。
  • 如果B的前两位为1，第三位为0，则该字符由两个字节组成，且B是第一个字节。
  • 如果B的前三位为1，第四位为0，则该字符由三个字节组成，且B是第一个字节。
  • 如果B的前四位为1，第五位为0，则该字符由四个字节组成，且B是第一个字节。
  • 空字符为0xC080。
